# 🚀 TaskFlow

TaskFlow est une application web simple de gestion de tâches développée avec PHP, MySQL et JavaScript. / TaskFlow is a simple web task management application developed with PHP, MySQL and JavaScript.

## ✨ Fonctionnalités / Features

- **Gestion des tâches** : Ajout, édition, suppression et marquage comme terminé. / **Task Management**: Add, edit, delete, and mark as completed.
- **Échéances & Alertes** : Support des dates avec indicateurs visuels (retard, imminent). / **Deadlines & Alerts**: Date support with visual indicators (overdue, imminent).
- **Recherche & Suivi** : Filtrage par mot-clé et barre de progression. / **Search & Tracking**: Keyword filtering and progress bar.
- **UI Moderne** : Glassmorphism, arrière-plan animé (parallaxe) et notifications toast. / **Modern UI**: Glassmorphism, animated background (parallax), and toast notifications.
- **Multilingue** : Support complet FR/EN. / **Multilingual**: Full FR/EN support.

## 🛠️ Technologies Utilisées / Technologies Used

- **Backend** : PHP 8.x, MySQL
- **Frontend** : JavaScript (Vanilla), HTML5, CSS3
